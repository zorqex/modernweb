# Kyokushinkai Karate — сайт клуба

Сайт для секции Киокушинкай каратэ. Сделан на чистом HTML/CSS/JS, без фреймворков.

**Живой сайт:** https://zorqex.github.io/modernweb/

---

## Что есть

- Главная с героем и статистикой
- Раздел о стиле и философии Ояма
- Программа тренировок (горизонтальный скролл)
- Переключатель тем — клик по блоку меняет цветовую схему всего сайта
- Система поясов
- Форма записи в секцию

## Запуск локально

```bash
python -m http.server 8000
# открыть http://localhost:8000
```

или просто открыть `index.html` в браузере.

## Стек

HTML5, CSS3 (custom properties, grid, flexbox), vanilla JS
