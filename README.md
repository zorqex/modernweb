## 🚀 Быстрый старт

### Просмотр сайта
Просто откройте `index.html` в браузере:
```bash
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```

### Локальный сервер
Для лучшего опыта используйте локальный сервер:
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (если установлен http-server)
npx http-server
```

Затем откройте: `http://localhost:8000`

## 🎯 Основные секции

- **Hero** - Вводная секция с параллакс фоном
- **Features** - 4 красивые карточки с возможностями
- **Horizontal Scroll** - Уникальное горизонтальное прокручивание
- **Gallery** - Галерея с 6 красивыми градиентами
- **Stats** - Счётчики с анимацией
- **Contact** - Форма обратной связи
- **Footer** - Завершающая информация
